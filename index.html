☁️ GitHub Sync

Your data will be saved to garden-data.json in your GitHub repo after every change. Survives phone upgrades, Safari clears, and any device.

github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → repo scope

💾 Local Backup

Download all your data as a JSON file. Keep it somewhere safe as a manual backup.

⚠️ Danger Zone

Wipes all local data and reloads app defaults. GitHub data is not affected — you can restore from GitHub afterwards. Only use this if something is seriously broken.
